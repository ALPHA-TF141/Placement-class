Two Pointer Linked List Problem - Solution

This problem asks you to reverse a linked list in-place using only two pointers.

            Given: Head = [1, 2, 3, 4]
            Return: [4, 3, 2, 1]

            Constraints:
                - No extra space allowed
                - Only modify the given nodes
                - Do not return a new list

            Example:
                Input: head = [1, 2, 3, 4]
                Output: head = [4, 3, 2, 1]

Implementation:

We use two pointers: prev and curr.

  1. prev starts at null and moves forward.
  2. curr starts at the head of the list.
  3. For each node, we: - Store the next node in a temporary variable. - Set the current node's next pointer to point to prev. - Move prev and curr one step forward.

Complexity:

O(n): One pass through the list.

O(1): Only constant extra space is used.

Explanation

            [1, 2, 3, 4]
              p   c
           prev = null
           curr = 1

           1's next points to prev (now null)
           prev = 1; curr = 2

           2's next points to prev (now 1)
           prev = 2; curr = 3

           3's next points to prev (now 2)
           prev = 3; curr = 4

Code:

            // Node definition
            function ListNode(val, next) {
                this.val = val === undefined ? 0 : val;
                this.next = next === undefined ? null : next;
            }

            // Function to reverse the linked list in-place
            function reverse(head) {
                let prev = null;  // Pointer to track the previous node
                let curr = head;   // Pointer to traverse the list

                while (curr !== null) {
                    // Store the next node
                    const temp = curr.next;

                    // Reverse the current node's pointer
                    curr.next = prev;

                    // Move pointers one step forward
                    prev = curr;
                    curr = temp;
                }

                return prev;  // New head of the reversed list
            }
